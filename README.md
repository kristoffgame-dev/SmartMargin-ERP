# 📊 Sistema de Control de Lotes, Márgenes y Stock

Una aplicación de escritorio profesional e intuitiva diseñada para emprendedores y comerciantes de mayoreo. Permite calcular el costo real unitario de mercancía importada o comprada en lote, gestionando mermas, costos de envío y proyecciones de ganancias, además de incluir un módulo seguro de ventas y reportes listos para imprimir en PDF.

## 🚀 Características Principales
* **Entrada Inteligente:** Filtro universal que acepta números con comas de miles y puntos decimales (`15,500.50` o `15500.50`).
* **Base de Datos Local:** Conexión robusta mediante SQLite para el almacenamiento permanente del historial.
* **Módulo de Ventas en Vivo:** Descuento de stock en tiempo real con alertas de inventario insuficiente.
* **Seguridad Financiera:** Exportación automática de reportes ejecutivos directamente a formato PDF no modificable a través del navegador.
* **Interfaz Limpia:** Diseñada con Tkinter bajo estándares de scannability y usabilidad.

## 🛠️ Tecnologías Utilizadas
* **Python 3**
* **Tkinter** (Interfaz Gráfica)
* **SQLite3** (Base de Datos Relacional)
* **HTML5/CSS3** (Formato de impresión del reporte)
